# Utilería ASCII

Utilería en java para generar tablas ASCII por consola.
trae un Main para ejecutarlo y ver un ejemplo
Ejemplo: 
    
    ```
        +----------+----------+----------+----------+
        | ID       | NOMBRE   | APELLIDO | EDAD     |
        +----------+----------+----------+----------+
        | 0        | BRYAN    | TIBAN    | 20       |
        +----------+----------+----------+----------+
        | 1        | BRYAN    | TIBAN    | 21       |
        +----------+----------+----------+----------+
        | 2        | BRYAN    | TIBAN    | 22       |
        +----------+----------+----------+----------+
        | 3        | BRYAN    | TIBAN    | 23       |
        +----------+----------+----------+----------+
        | 4        | BRYAN    | TIBAN    | 24       |
        +----------+----------+----------+----------+
        |
        +----------+----------+----------+----------+
    ```

## Tecnologías

- Java 8
- Maven 3.8.1
